# Job Searching Bot — WebWork Scout

Automated freelance order intelligence for a 3-agent website delivery pipeline.

## The Pipeline

```
SCOUT (Research)  →  BUILDER (Build)  →  DEPLOYER (Ship)
        ↓                  ↓                    ↓
  briefing.md       /output/site/         surge.sh URL
```

Typical turnaround: **1–2 days** from brief to live preview URL.

## Order Targeting

- **Budget:** $40 – $3,000 (sweet spot $150–$1,500)
- **Stack:** HTML5 / CSS / Vanilla JS — no CMS, no frameworks
- **Scope:** Static websites, landing pages, business sites, redesigns
- **Timeline:** Half-day to 7 days depending on size

## Key Finding (March 2026)

80%+ of web design orders on mainstream freelance platforms specify WordPress,
Shopify, or Squarespace. The team's vanilla static stack is a positioning
challenge. Priority sourcing tactic: target clients with Lovable/Bubble/Wix
sites who want to migrate to a custom coded, SEO-clean static site.
