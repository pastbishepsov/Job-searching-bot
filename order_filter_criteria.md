# Order Filter Criteria

## ACCEPT if ALL of these are true:

1. **No platform lock-in** — client has not specified WordPress, Shopify, Wix, Squarespace, Webflow, or any CMS as a hard requirement
2. **Static site deliverable is viable** — output can be HTML/CSS/JS files hosted on any server
3. **Scope fits** — brochure/marketing/lead-gen site, not a web application with auth/data/backend
4. **Budget ≥ $40**
5. **Client has a brief or reference** — SCOUT needs something to research (existing site, business name, industry, or brief)
6. **Not already awarded / expired**
7. **Proposal count manageable** — under ~50 bids preferred; 80+ bids = unwinnable at standard pricing

## REJECT immediately if ANY of these apply:

- Requires WordPress, Shopify, Wix, Squarespace, Webflow, Elementor, Divi, WooCommerce
- Requires backend logic: user auth, databases, order tracking, payment processing
- Requires complex integrations: Stripe, HubSpot API, custom REST API, SMS gateway
- Requires a CMS admin panel
- Budget is a test placeholder ($1–$10 on Upwork = almost always fake)
- Post is already awarded or cancelled
- Proposal count 80+ with no price signal advantage
- Requires React/Vue/Angular/Next.js product UI

## Platform-specific judgment calls:

| Platform mentioned | Action |
|-------------------|--------|
| "WordPress" | Reject unless client says platform is flexible |
| "Hostinger" | Investigate — could be their drag-and-drop builder (reject) OR just their hosting (accept) |
| "HTML/CSS/JS" explicitly | Accept immediately |
| "no preference on stack" | Accept |
| "Lovable / Bubble / Wix" → wants to migrate to coded site | High-priority accept |
| "Figma file exists" | Strong positive signal — scope is bounded |
| "from scratch" with no platform mentioned | Accept |

## Scoring weights for ranked comparisons:

| Criterion | Weight |
|-----------|--------|
| Budget adequacy (≥$40, sweet spot $150–$1,500) | 30% |
| Scope clarity (existing brief, reference site, page list) | 25% |
| Static-site deliverable compatibility | 20% |
| Client responsiveness signals (recent activity, replies, track record) | 15% |
| Competitive pressure (fewer bids = better odds) | 10% |
